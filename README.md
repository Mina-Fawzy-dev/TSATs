## TSATs (Technical Support Automation Tools)

TSATs is a growing collection of automation tools, scripts, and utilities designed to simplify common IT Support and Help Desk tasks. This repository showcases practical solutions for troubleshooting, system administration, diagnostics, repetitive support processes, and everyday IT operations.

The goal is to build real-world tools that improve efficiency, reduce manual work, and demonstrate practical technical support, automation, and problem-solving skills across different technologies.
